# ✦ eugrifo

**Destaque textos na web e exiba tudo em um widget --- sem servidor, sem
backend.**

![license](https://img.shields.io/badge/license-MIT-lightgrey)

## ✦ Visão geral

O **eugrifo** é uma extensão que permite salvar trechos destacados no
GitHub e exibi-los como um widget em qualquer site.

-   💾 Armazena no GitHub\
-   ⚡ Zero backend\
-   🌐 Funciona em qualquer página\
-   🎨 Personalizável

## ✦ Pré-requisitos

-   Conta no GitHub\
-   Um repositório\
-   Navegador Chromium\
-   (Opcional) Um site

## ✦ 1. Criar repositório no GitHub

1.  Acesse https://github.com\
2.  Clique em New repository\
3.  Nomeie (ex: meus-highlights)\
4.  Escolha Public ou Private\
5.  Add README\
6.  Create repository

## ✦ 2. Gerar token

1.  Settings → Developer settings → Tokens\
2.  Generate new token\
3.  Nome: eugrifo-extension\
4.  Expiração: No expiration\
5.  Scope: repo\
6.  Copie o token

## ✦ 3. Baixar e descompactar

Arquivo: eugrifo.zip

Windows: botão direito → Extrair tudo\
Mac: duplo clique\
Linux: unzip eugrifo.zip

## ✦ 4. Instalar

chrome://extensions\
Ativar modo dev\
Carregar sem compactação\
Selecionar pasta

## ✦ 5. Configurar

Token / Usuário / Repo / Arquivo

## ✦ 6. Highlight

Selecionar texto → salvar

## ✦ 7. Widget

```{=html}
<script src="https://cdn.jsdelivr.net/..."></script>
```
